# Тестирование

Проект использует многоуровневый подход к проверке кода для обеспечения стабильности системы.

## Контроль качества (CI/CD)

- **Frontend (React, TypeScript, Vite)**
  - Статический анализ и форматирование: Prettier (`npx prettier --check .`) и ESLint (`npx eslint .`).
  - Проверка типов: `npm run typecheck`.
  - Выполнение unit-тестов компонентов и служебных функций: `npm test`.
  - Проверка успешности сборки: `npm run build`.

- **Gateway API (Node.js)**
  - Линтинг: ESLint.
  - Проверка типов и выполнение тестов маршрутизатора API: `npm test`.

- **OCR сервис (Python, FastAPI)**
  - Контроль качества кода: `flake8`, `black --check`, `ruff check`.
  - Unit и интеграционные тесты логики распознавания: `pytest`.

- **Среда развертывания (Docker)**
  - Валидация и сборка конфигураций `docker compose`.
  - Мониторинг стабильности контейнеров (healthchecks).
